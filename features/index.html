Bot Features

Here you can find a complication of select features and commands available to you through Levante.

Features

Things that the bot will do for you.

  • XP Logging: Generally used for AFK XP farming using whatever method of your choice.
    • Server Administrators (specificially those with the Manage Channels permission) can use the command /create-hub to create a channel category where server members can start logging their XP.
    • Server members can then, use the "XP Hub" and its buttons to manage their XP Sessions accordingly.
  • Emblem Offers: An announcement feature where the bot will notify server members of any newly released, limited time emblems.
    • Server Administrators can opt-in via the /alert emblem-offers command in the channel they want these alerts in.
  • Rotation Alerts: Using the command /notify will have the bot message you on Discord when your specified rotation is active.
    • Only supported rotations can be tracked.

Supporter Perks

Additional perks available to Supporters.

  • Priority XP Logging: Users using priority XP logging get refreshed first, and the ability to bypass the capacity limit.
    • Users are placed into priority automatically when they become a supporter.
  • Exclusive Clan Tag: Users that use the /try-on command will have the "Levante Bot Supporter" clan tag as opposed to "Levante Bot".
    • Flex any emblem as well as your supporter status with us.
  • Custom Role: A personalized role and access to a super-secret channel in the Levante Support Server.
    • Once you donate, you'll be able to notify any of the staff members about your generous support and they can get you situated.
